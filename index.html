About Me



Kai Hao

I first started to learn programming in freshman course ”Programming Design,” but actually I have been interested in programming since I was a junior high school student. What the programming languages has brought me the most is that it makes me believe I can change the world by myself and through my own thinking. I did very well in the course learning C++ in my first year of college and I often taught my classmate about our assign- ment. Later in my second and third year of college, I tried to learn a lot of other languages. I learned Java for Android app design and C# for windows .NET platform application. Then I tried to learn a lot of languages like Python and Ruby.

In my third year of college, I started to learn basic web design for the ”Database” course project. During only two weeks of learning from zero knowledge of web design, I tried PHP and MySQL along with Apache, using HTML, CSS, and Javascript framework JQuery for front-end design for only two days. I built a website for viewing database and basic CRUD operations in prototype version on my own. Then I spent another two days to change my website from JQuery to AngularJS. Then I changed the LAMP(stands for Linux, Apache, MySQL, PHP) to MEAN(MongoDB, ExpressJS, AngularJS, NodeJS) for another two days. Finally I spent the rest of the days implementing features and optimizing the web- site with my two group mates. This experience was my first time to learn so many languages and programming skills in such a short time, and I really love learning new things and gain the sense of accomplishment by doing it.

During my learning process, I find that I am more interested in front-end design and mobile application design. The idea that I can build my own application with great collabo- rators and learn skills from them makes me very excited, and I’m looking forward to work or intern for great companies and share the learning with my colleagues.
